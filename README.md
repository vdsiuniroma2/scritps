# scritps
Useful scripts for VDSI course

## `install_vdsi_tools.sh`
This script can be used to _weaponize_ a Debian/ubuntu based distro.<br>
Software that need to be installed:

- General networking tools
    * `curl`
    * `dig`
    * `ghidra`
    * `host`
    * `netcat`
    * `wget`

- Enumeration
    * `dirbuster`
    * `gobuster`
    * `nmap`

- Web VAPT tools
    * `burp`
    * `hydra`

- Password cracking
    * `Hashcat`
    * `John the Ripper`

- Wordlists
    * `rockyou`
    * `seclists`
